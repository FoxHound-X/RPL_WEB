# Aplikasi Sekolah Berbasis Web

Aplikasi ini adalah Tugas Yang di buat untuk memudahkan sekolah memanagement Data sekolah
Seperti Absensi, Jurnal Guru, dan lain nya

<br>
<br>

## Fitur Aplikasi

- Dashboard Admin
- Absensi Siswa
- jurnasl Untuk Guru
- Rekap Pembayaran SPP [Fitur Yang Kami Kerjakan Saat Ini]
- Login & Logout
